# CmpInstall
The source code of Sir Henry's Anno 1602 Campaign Installer

The tool is used to create additional campaigns for the game Anno 1602 (1602 A.D.). It is available in the [Annopool](https://www.annozone.de/annopool/file/948-sir-henry-s-anno-1602-kampagnen-installer/). A campaign consists of up to 5 scenarios to be played in sequence.
